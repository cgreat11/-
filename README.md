# 节假日数据字典说明

该目录下存放了各年份的日期数据（如 `2021.json` 等）。这些 JSON 数据记录了每一天的基础日期信息以及当天的“工作/休息”属性。
> **说明：**
> 法定节假日数据已更新到 **2026** 年。后续年份的数据需根据国务院每年发布的法定节假日安排进行更新。
以下是 JSON 结构中各个字段的名称及其数据字典的详细说明：

## 字段说明

| 字段名称 | 类型 | 说明 | 字典及取值范围 |
| :--- | :--- | :--- | :--- |
| `nian` | Integer | 年份 | 例如：`2021`、`2022` 等 |
| `yue` | Integer | 月份 | `1` ~ `12` |
| `ri` | Integer | 日期（日） | `1` ~ `31` |
| `xq` | Integer | 星期 | `1`: 星期一<br>`2`: 星期二<br>`3`: 星期三<br>`4`: 星期四<br>`5`: 星期五<br>`6`: 星期六<br>`7`: 星期日 |
| `rqlx` | Integer | 日期类型 | **`1`**: 工作日（包含正常的周一至周五，以及因法定节假日调休而需要上班的周末补班）<br>**`2`**: 休息日/节假日（包含正常的周末双休日，以及法定节假日放假） |
| `rq` | String | 完整的日期字符串 | 格式为 `YYYY-MM-DD`，例如：`2021-01-01` |

## 数据示例

```json
{
    "nian": 2021,
    "yue": 1,
    "ri": 1,
    "xq": 5,
    "rqlx": 2,
    "rq": "2021-01-01"
}
```
